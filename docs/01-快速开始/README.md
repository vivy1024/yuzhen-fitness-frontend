## 🎯 学习路径

1. **环境准备** (10分钟)
   - 安装Node.js 18+
   - 安装pnpm
   - 配置VS Code

2. **项目启动** (5分钟)
   - 克隆代码
   - 安装依赖
   - 启动服务

3. **熟悉项目** (30分钟)
   - 了解目录结构
   - 查看核心组件
   - 理解数据流

4. **开始开发** (1小时)
   - 创建新组件
   - 添加新页面
   - 对接API
