## 📱 账号相关

### Q1: 如何注册账号？

**A**: 玉珍健身支持多种注册方式：

1. **手机号注册**
   - 打开应用，点击"注册"按钮
   - 输入手机号，获取验证码
   - 设置密码（至少8位，包含字母和数字）
   - 完成注册

2. **邮箱注册**
   - 选择"邮箱注册"
   - 输入邮箱地址
   - 查收验证邮件，点击链接激活
   - 设置密码完成注册

3. **第三方登录**（即将推出）
   - 支持微信、QQ快速登录
   - 首次登录需补充基本信息

### Q2: 忘记密码怎么办？

**A**: 可以通过以下方式找回密码：

1. 在登录页面点击"忘记密码"
2. 输入注册时使用的手机号或邮箱
3. 获取验证码
4. 设置新密码
5. 使用新密码登录

**注意**: 验证码有效期为10分钟，请及时使用。

### Q3: 如何修改个人信息？

**A**: 修改个人信息步骤：

1. 进入"我的"页面
2. 点击头像或"编辑资料"
3. 可修改的信息包括：
   - 昵称
   - 头像
   - 性别
   - 生日
   - 身高体重
   - 健身目标
4. 修改后点击"保存"

**提示**: 身高、体重等数据会影响AI训练计划的生成，建议保持更新。

### Q4: 如何保障账号安全？

**A**: 我们提供多重安全保障：

1. **密码安全**
   - 使用强密码（至少8位，包含字母、数字、特殊字符）
   - 定期更换密码
   - 不要在多个平台使用相同密码

2. **登录保护**
   - 异地登录提醒
   - 可查看登录历史
   - 支持强制下线其他设备

3. **数据加密**
   - 密码采用bcrypt加密存储
   - 通信使用HTTPS加密
   - 敏感数据本地加密存储

4. **隐私设置**
   - 可设置训练记录可见性
   - 可关闭数据分享功能

### Q5: 如何注销账号？

**A**: 账号注销流程：

1. 进入"我的" → "设置" → "账号与安全"
2. 点击"注销账号"
3. 阅读注销须知
4. 输入密码确认
5. 等待7天冷静期
6. 账号正式注销

**重要提示**:
- 注销后所有数据将被永久删除，无法恢复
- 会员权益将立即失效，不予退款
- 建议先导出训练数据备份

## 💪 训练相关

### Q1: 如何使用AI生成训练计划？

**A**: AI训练计划生成步骤：

1. **完善个人档案**
   - 进入"我的" → "个人档案"
   - 填写身高、体重、年龄、性别
   - 选择健身目标（增肌/减脂/塑形/力量）
   - 填写训练经验和可用器械

2. **生成计划**
   - 进入"AI助手"页面
   - 描述你的训练需求，例如：
     - "我想练胸肌，有哑铃和杠铃"
     - "帮我设计一个减脂计划"
     - "我是新手，想开始健身"
   - AI会基于你的档案和需求生成个性化计划

3. **计划内容**
   - 包含动作选择、组数次数、重量建议
   - 考虑肌群平衡和恢复时间
   - 提供安全提醒和注意事项

**提示**: 档案信息越完整，AI生成的计划越精准。

### Q2: 动作库如何使用？

**A**: 动作库功能详解：

1. **浏览动作**
   - 进入"动作库"页面
   - 按肌群分类浏览（胸、背、腿、肩、臂等）
   - 按器械筛选（杠铃、哑铃、固定器械、自重等）
   - 按难度筛选（新手、中级、高级）

2. **动作详情**
   - 动作名称（中英文）
   - 目标肌群和协同肌群
   - 动作描述和技术要点
   - 安全提醒
   - 常见错误
   - 替代动作建议

3. **收藏动作**
   - 点击动作卡片的"收藏"按钮
   - 在"我的收藏"中查看
   - 快速添加到训练计划

4. **搜索功能**
   - 支持中英文搜索
   - 支持模糊匹配
   - 支持按肌群、器械、难度组合筛选

**数据规模**: 当前动作库包含1,790个专业健身动作，覆盖所有主要肌群。

### Q3: 如何记录训练？

**A**: 训练记录功能：

1. **开始训练**
   - 选择训练计划或自由训练
   - 点击"开始训练"

2. **记录过程**
   - 每完成一组，记录：
     - 重量（kg）
     - 次数
     - 完成质量（可选）
   - 支持快速复制上一组数据
   - 支持添加备注

3. **完成训练**
   - 点击"完成训练"
   - 系统自动统计：
     - 训练时长
     - 总训练量
     - 肌群分布
   - 可添加训练感受和照片

4. **查看历史**
   - 在"训练记录"页面查看历史
   - 支持按日期、肌群筛选
   - 查看进步曲线和数据分析

### Q4: 训练计划可以自定义吗？

**A**: 当前版本支持AI生成计划，用户自建计划功能即将推出：

**当前功能**:
- AI根据需求生成个性化计划
- 可以在训练中调整重量和次数
- 可以替换动作

**即将推出**（v2.0）:
- 从动作库自主创建计划
- 保存和分享自定义计划
- 计划模板库
- 周期化训练设计

### Q5: 如何查看训练进度？

**A**: 进度追踪功能：

1. **数据统计**
   - 进入"我的" → "训练统计"
   - 查看：
     - 总训练天数
     - 总训练时长
     - 总训练量
     - 肌群训练分布

2. **进步曲线**
   - 查看各动作的重量进步
   - 查看体重变化趋势
   - 查看体脂率变化（需手动记录）

3. **成就系统**（即将推出）
   - 训练里程碑
   - 徽章收集
   - 排行榜

### Q6: AI助手可以回答哪些问题？

**A**: AI助手功能范围：

1. **训练计划**
   - 生成个性化训练计划
   - 调整现有计划
   - 动作替代建议

2. **动作指导**
   - 动作技术要点
   - 常见错误纠正
   - 安全注意事项

3. **营养建议**
   - TDEE计算
   - 营养摄入建议
   - 膳食计划设计

4. **问题解答**
   - 健身知识科普
   - 训练疑问解答
   - 损伤预防建议

**技术支持**: 基于DAML-RAG框架，结合Neo4j知识图谱（4,246节点，61,507关系）和专业运动学知识库。

- 数据经过专业审核

### Q5: 数据会被删除吗？

**A**: 数据保留政策：

**正常使用**:
- 数据永久保留
- 定期自动备份
- 不会主动删除

**账号注销**:
- 7天冷静期内可撤销
- 冷静期后数据永久删除
- 删除前可导出备份

**长期未登录**:
- 1年未登录：发送提醒邮件
- 2年未登录：账号冻结，数据保留
- 3年未登录：账号注销，数据删除

**数据备份建议**:
- 定期导出重要数据
- 保存到本地或云盘
- 避免依赖单一平台
