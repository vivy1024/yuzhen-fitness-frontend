## 玉珍是谁

**玉珍**，是我奶奶的名字。

冯玉珍，1943年10月11日出生于西安长安区樊川冯家村，2021年8月9日因脑溢血离世，享年79岁。

她是一位典型的中国农村女性——一生劳作、起早贪黑，为家庭付出了全部。从年少时为弟弟吃苦，到出嫁后养育三子，再到晚年仍在田间劳作、还债养家，她的一生是无数中国农村女性的缩影。

即便在古稀之年，她依然为我们这些孙辈操劳，从不言苦。她的坚韧、勤劳和无私的爱，深深影响着我。

## 我的健身历程

我从2021年10月开始自学健身，那时的我只有100斤。

经过四年的坚持训练，我从100斤练到了130斤，实现了：
- **深蹲**: 125kg
- **硬拉**: 130kg  
- **卧推**: 80kg

健身不仅改变了我的体型，更改变了我的生活态度。在训练中，我学会了坚持、自律和科学规划。

2025年8月，在使用AI辅助开发的过程中，我萌生了创建训练记录软件的想法——一个能帮助更多人科学健身的智能应用。

## 为什么叫玉珍健身

起名时，我想到了奶奶。

奶奶一生都希望家人身体健康。她总是说："身体好比什么都强。"

如今，奶奶已经离开了。但我希望，身体好的奶奶，也能看见孙子身体好了、快乐了。

**玉珍健身**，不只是一个健身应用，更是我对奶奶的纪念，一种"身体好了"的证明。

每一次训练记录，都是对奶奶的思念；  
每一次进步，都是对她期望的回应。

## 《樊杜有冯老》

以下是我为奶奶所作的五言叙事诗，记录了奶奶一生的真实经历。

此诗参考《孔雀东南飞》《木兰辞》等汉乐府叙事诗风格，按时间线分章节叙述。


### 序章·樊川冯氏

```
樊川有冯氏，玉珍是其名。
癸未秋月生，长安故土情。
生来为长女，幼弟待其成。
年少知艰辛，苦难伴此生。
```

### 第一章·少年出嫁

```
韶华十七载，嫁与秉为妻。
夫妻同劳作，日出而耕犁。
三年饥荒至，天灾降人间。
秉郎苦奔波，麸糠换命钱。

腹中怀长子，饥寒交迫时。
无米亦无面，苦水度朝夕。
长子诞辰日，绝症随之来。
苦难缠冯老，命运多乖违。
```

### 第二章·三子成家

```
辛苦得二子，荒年终推移。
二儿力气大，众人皆称奇。
虽然早肄业，心中有主意。
赚钱养家门，父母得安宜。

后又得三子，天资异常人。
寺坡称神童，算术胜机器。
冯老何欢喜，三儿最宠溺。
可叹伤仲永，难称如人意。

肄业走天下，无银便取资。
冯老护三儿，终使其性逸。
二儿事业成，娶妻生大川。
孙辈呈川字，海纳百川兮。
```

### 第三章·苦难连连

```
辛勤为二儿，盖得婚房矣。
千禧年未至，二儿病魔欺。
治病何得钱，东借西要噫。
何其悲伤哉，何其劳累兮。

梁柱倒塌陷，家境变难矣。
大儿累其病，从不事农桑。
三儿其年小，难以承其忙。
苦难累冯老，花甲种地秧。

农闲不得息，款债须还乡。
白日做蜡烛，晚来拾夜苨。
二儿逝年许，愿三儿冲喜。
三儿婚成兮，得小川为子。
```

### 第四章·晚年劳作

```
冯老何欢喜，老来享两孙。
时日不平和，大儿绝症起。
苦难刹冯老，白发送黑发。
得年六十五，失子二其一。

前债未清兮，又添大儿债。
终日不得闲，使其意志毅。
上管秉作活，下管孙学习。
性情终不改，愿家不得息。

小川再得宠，惯出小霸王。
日日顶撞姥，丧其开心颜。
大小事必较，愿小川安康。
若得小川在，鞠躬尽瘁死。

后得其三孙，随其娘家姓。
冯老不诧异，更欢喜得孙。
本为一古稀，愿其养天年。
冯老志不停，愿为其孙劳。
其孙何得此，终日嫌其烦。
```

### 第五章·辛丑之变

```
辛丑年来至，冯老何欢喜。
大川成其婚，小川耀门楣。
本为天年乐，得享颐养福。
彩礼需其钱，学业要其钱。

是乎不食药，言为须省钱。
呜呼悲乎哉，血压高不止。
血冲其脑内，大小便失禁。
身体不受控，意识渐模糊。

家人突发现，何其悲伤矣。
小川唤医车，又唤其大哥。
大哥悲欲绝，一路泪满襟。
三儿伤心智，三儿媳尽责。

秉无言端坐，床头人不再。
三孙方年小，不知何事生。
小川却不悲，脑内乱绪起。
苦难终冯老，享寿七十九。
```

### 尾声·思念与传承

```
月许复年许，小川才得感。
睹物方思人，才知人其贵。
当时月正圆，却再无个人，
耄耋劝入眠，劝入眠！

奶奶常言道，身体好最强。
如今孙已壮，百斤变百三。
深蹲一百二，硬拉一百三。
卧推八十斤，健体慰九泉。

玉珍为名号，健身为其业。
不只是应用，更是思念结。
每次记训练，都是念奶奶。
每次有进步，都是报期盼。

愿奶奶在天，能见孙安康。
身体好了矣，快乐亦常伴。
玉珍健身者，传承爱无边。
```

## 诗歌创作说明

这首诗参考了汉乐府叙事诗《孔雀东南飞》《木兰辞》的风格特点：

1. **韵律**：以五言为主，偶句押韵
2. **叙事**：按时间线叙述，保留所有真实事件
3. **情感**：在叙事中融入情感，结尾升华
4. **结构**：序章→五章→尾声，共七个部分

诗中记录的所有事件均为真实经历，包括：
- 1943年奶奶出生于樊川冯家村
- 三年自然灾害时期的艰难
- 养育三子的辛劳
- 我二伯、大伯相继离世的悲痛
- 奶奶花甲之年仍在劳作还债
- 2021年辛丑年奶奶因脑溢血离世
- 我从100斤练到130斤的健身历程
- 玉珍健身应用的诞生
