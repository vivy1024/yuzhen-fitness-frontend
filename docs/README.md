## 🔧 技术栈

### 核心框架
- **Vue 3.4**: Composition API、`<script setup>`
- **TypeScript 5.3**: 类型安全
- **Vite 5.0**: 快速构建工具

### 状态管理
- **Pinia 2.1**: 现代化状态管理
- **Pinia Persist**: 状态持久化

### UI组件
- **shadcn-vue**: 基于Radix UI的组件库
- **Tailwind CSS 3.4**: 原子化CSS
- **lucide-vue-next**: 图标库

### 路由和请求
- **Vue Router 4.2**: 路由管理
- **Axios**: HTTP客户端

### PWA
- **Vite PWA Plugin**: PWA支持
- **Workbox**: Service Worker

### 开发工具
- **ESLint**: 代码检查
- **Prettier**: 代码格式化
- **TypeScript**: 类型检查
